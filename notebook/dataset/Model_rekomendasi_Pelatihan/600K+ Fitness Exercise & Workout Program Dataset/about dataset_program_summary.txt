📊 Program Summary (2,598 unique fitness programs)
This companion file provides aggregated metadata for 2,598 distinct fitness programs, offering a high-level view of the workout collection. Each row represents a complete fitness program with summarized characteristics.

🔑 Key Features:

Program overview: title, description, and target fitness level
Goal-specific targeting: muscle building, weight loss, strength, etc.
Equipment requirements and program duration
Workout timing: average time per workout session
Program scope: total number of exercises included
Temporal metadata: creation and last modification dates
📝 Data Dictionary:

title: Program name (2,598 entries)
description: Program overview and objectives (2,594 entries)
level: Target fitness level (beginner/intermediate/advanced)
goal: Primary fitness objective
equipment: Required equipment type (2,597 entries)
program_length: Total program duration in weeks (2,597 entries)
time_per_workout: Average workout duration in minutes
total_exercises: Total number of exercises in the complete program
created/last_edit: Program creation and modification timestamps
🎯 Use Cases:

Program-level analysis and clustering
Recommendation system training at the program level
Understanding fitness program popularity and characteristics
High-level workout planning and categorization
Program effectiveness analysis based on structure and goals
This summary dataset complements the detailed exercise data by providing program-level insights for broader analysis and recommendation system development.
