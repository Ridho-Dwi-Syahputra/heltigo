🏋️ About This Dataset
This comprehensive fitness dataset contains over 600,000 structured workout routines and exercise entries scraped from fitness planning platform data. The dataset is designed for building recommendation systems, analyzing workout patterns, and understanding fitness program structures.

📊 Dataset Overview

605,033 individual exercise entries
17 comprehensive features covering workout structure, exercise details, and program metadata
Spans different fitness levels (beginner, intermediate, advanced)
Covers various fitness goals (strength, cardio, weight loss, muscle building)
Includes equipment requirements and workout timing information
🔑 Key Features

Exercise details: name, sets, reps, intensity
Program structure: week, day, time per workout
User targeting: fitness level, goals, equipment needs
Temporal data: creation and edit timestamps
Program metadata: length, number of exercises per workout
🎯 Use Cases

Building workout recommendation systems
Analyzing fitness program effectiveness
Understanding exercise patterns and trends
Creating personalized workout generators
Fitness app development and research
🔧 Technical Details

Format: CSV
Size: ~296MB
Data quality: Minimal missing values (<1% for most columns)
Collection period: 01/07/2025
Source: boostcamp.app
📝 Data Dictionary

title: Workout/program name
description: Detailed workout description
level: Fitness level (beginner/intermediate/advanced)
goal: Primary fitness objective
equipment: Required equipment type
program_length: Duration in weeks
time_per_workout: Duration per session (minutes)
week/day: Position in program structure
number_of_exercises: Total number of exercises in the programm
exercise_name: Specific exercise name
sets/reps: Exercise volume parameters (negative reps values mean time in seconds)
intensity: Exercise intensity level (0-10)
